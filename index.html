VisDice - Visualisation Design Ideation

A web version of the VisDice application for design thinking and ideation. Click to roll each die or roll them all at once.

Program: visDiceApp.

Application to run the VisDice design method of design thinking. The program runs in the Processing.org environment. The program loads images from file and allows people to select and "roll" the images Images can be rolled one by one, or all together. Images can be shown (on) or hidden (off).
